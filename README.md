# cxliu0314.github.io

Changxing Liu 的个人学术主页，基于 Rui Ye 主页模板改造。

## 本地预览

```powershell
cd D:\Workspace\personal_homepage
python -m http.server 8000
```

然后打开 `http://localhost:8000`。

## 内容

- `index.html`：主页主体内容。
- `stylesheet.css`：Rui Ye 模板样式。
- `images/`：头像、论文图片和 SJTU 标志。
- `PERSONAL_INFO_FORM.md`：信息填写表，保留用于后续维护。

## 部署

推送到 `main` 分支后，通过 GitHub Pages 访问：

`https://cxliu0314.github.io/`
